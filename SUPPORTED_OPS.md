## Ethos-U55 and Ethos-U65 TFLite Generic Constraints

This is a list of constraints that most operators must satisfy in order to be scheduled on the NPU.  
(Operators excluded from certain constraints are listed as exceptions )

- Feature-map dataTypes must be one of {"INT16", "INT32", "INT64", "INT8", "UINT8"}
- Operations must have at least one IFM.
- Operations must have at least one OFM.
- Tensors must have constant shape.
- Only the first dimension of tensors can be dynamic.
- Input(s), Output and Weight tensors must have quantization parameters.
  - Exceptions: [ARGMAX, EXPAND_DIMS, MIRROR_PAD, PAD, PADV2, QUANTIZE, TRANSPOSE]
- Quantization scale of any tensor must be positive.
  - Exceptions: [ARGMAX, EXPAND_DIMS, MIRROR_PAD, PAD, PADV2, QUANTIZE, TRANSPOSE]
- Per-axis quantization is not supported.
  - Exceptions: [CONV_2D, DEPTHWISE_CONV_2D, FULLY_CONNECTED, TRANSPOSE_CONV]
- Zero-points must be 0 for Int32 and non-negative for unsigned dataTypes
  - Exceptions: [AVERAGE_POOL_2D, RESIZE_BILINEAR, RESIZE_NEAREST_NEIGHBOR, UNIDIRECTIONAL_SEQUENCE_LSTM]
- 64-bit feature-maps are not supported.
  - Exceptions: [ARGMAX]

This is a list of constraints that the AVERAGE_POOL_2D operator must satisfy in order to be scheduled on the NPU.

- 32-bit feature-maps are not supported.
- Kernel constraints:
  - IF padding=VALID:
    - Kernel product must be less than or equal to 256*256

### Ethos-U55 and Ethos-U65 TFLite TRANSPOSE Constraints

  - NHWC: C <= 2^16
  - NWHC: N ==1, H <= 2^16, W <= 2^16, C <= 2^14
  - NHCW: N*H <= 2^16, W <= 2^16, C <= 2^16
  - Any other permutation vector is unsupported
- IF IFM is Int8 or Int16
  - NHWC: no shape constraints
  - ELSE IF Rank <= 4D and permutation is: NWHC/NHCW/NCWH/NWCH/NCHW:
    - (N*H, W, C) <= (2^16, 2^16, 2^16)
  - ELSE:
    - Product of elements must be less than or equal to 2^16.
- Parameter tensors must be constant
- Tensor dimension must be <= 8

## Ethos-U85 TFLite Generic Constraints

This is a list of constraints that most operators must satisfy in order to be scheduled on the NPU.  
(Operators excluded from certain constraints are listed as exceptions )


- Feature-map dataTypes must be one of {"BOOL", "INT16", "INT32", "INT64", "INT8", "UINT8"}
- Operations must have at least one IFM.
- Operations must have at least one OFM.
- Tensors must have constant shape.
- Only the first dimension of tensors can be dynamic.
- Input(s), Output and Weight tensors must have quantization parameters.
  - Exceptions: [ARGMAX, EXPAND_DIMS, GATHER, MIRROR_PAD, PAD, PADV2, QUANTIZE, REDUCE_ALL, REDUCE_ANY, SCATTER_ND, SELECT, SELECT_V2, TRANSPOSE]
- Quantization scale of any tensor must be positive.
  - Exceptions: [ARGMAX, EXPAND_DIMS, GATHER, MIRROR_PAD, PAD, PADV2, QUANTIZE, REDUCE_ALL, REDUCE_ANY, SCATTER_ND, SELECT, SELECT_V2, TRANSPOSE]
- Per-axis quantization is not supported.
  - Exceptions: [CONV_2D, DEPTHWISE_CONV_2D, FULLY_CONNECTED, TRANSPOSE_CONV]
- IFM Zero-points must be in the range:
  - Int8: (-128 <= zp <= 127).
  - UInt8: (0 <= zp <= 255).
  - UInt16: zp must be 0 or 32768.
  - Other: zp must be 0.
  - Exceptions: [AVERAGE_POOL_2D, DIV, RESIZE_BILINEAR, RESIZE_NEAREST_NEIGHBOR, UNIDIRECTIONAL_SEQUENCE_LSTM]
- OFM Zero-points must be in the range:
  - Signed-integer: (-128 <= zp <= 127)
  - Unsigned-integer: (0 <= zp <= 255) OR zp == 32768.
  - Exceptions: [AVERAGE_POOL_2D, DIV, RESIZE_BILINEAR, RESIZE_NEAREST_NEIGHBOR, UNIDIRECTIONAL_SEQUENCE_LSTM]
